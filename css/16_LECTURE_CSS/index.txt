 🌟 Block-level element

 Some common examples of block-level elements in HTML are:

1. <div>: A generic container element used to group other HTML elements.
2. <p>: Represents a paragraph of text.
3. <h1> to <h6>: Heading elements used to define different levels of headings.
4. <ul>: Represents an unordered list.
5. <ol>: Represents an ordered list.
6. <li>: Represents a list item within an unordered or ordered list.
7. <table>: Represents tabular data.
8. <form>: Represents a form for user input.
9. <header>, <footer>, <nav>, <article>, <section>: Semantic elements used for structuring different parts of a webpage.
